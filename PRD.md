# 산단파크 (SANDAN PARK) — PRD

> **"산업단지에서 산다"** — 청년 근로자의 외로움을 레벨업으로 바꾸는 온·오프 믹스 RPG 플랫폼

---

## 1. Why — 문제 정의 (심사기준: 문제정의)

### 1-1. 김산단의 하루 (페르소나)
- 28세, 남성, 구미산단 제조업 3년차
- 외지에서 혼자 와서 1인가구, 기숙형 원룸 거주
- 퇴근 후 혼술·게임·잠의 반복
- "내가 뭘 모르는지조차 모르는 상태"
- 이직만이 탈출구라고 느낌

### 1-2. 구조적 문제 (2025 리빙랩 검증 결과)
공통 페인포인트 — **구미 · 창원 · 완주** 세 산단 모두에서 반복됨:
1. **외로움** — 퇴근 후 만날 사람 없음, 느슨한 연결 부재
2. **갈 곳 없음** — 공원, 도서관, 심야식당, 러닝 코스 부재
3. **놀 줄 모름** — 취미 탐색 경험 없음, 레퍼런스 없음
4. **고립된 인프라** — 블라인드는 대기업 중심, 소모임 앱은 도시 중심
5. **브랜드 이미지** — 산단 = 칙칙·삭막·일만 하는 곳

### 1-3. 우리의 관점 전환
> **일터에서 삶터로. 회색 공장에서 RPG 테마파크로.**

배틀그라운드 맵의 삭막함에서 리소스를 주워 미션을 수행하면 자기효능감이 생기듯,
산단의 풍경 자체를 **획득·성장·보상의 게임 공간**으로 재해석한다.

---

## 2. Who — 타겟 (심사기준: 필요성)

### 2-1. 주 타겟
- **전국 산업단지 청년 근로자** (20~30대)
- 비수도권, 1인가구, 중소기업, 외지 유입자 비중 높음
- 2025 기준 전국 산단 1,330개 → 2027 문화친화 산단 10개 로드맵

### 2-2. 부 타겟
- 산단공단 / 지자체 (정책 파트너)
- 지역 상권 (카페, 식당, 미용실 등 SDC 제휴)
- 로컬 크리에이터 (퀘스트·이벤트 호스트)

### 2-3. 왜 필요한가
- 블라인드: 대기업 전용 → 중소 산단 근로자 소외
- 소모임/프립: 도시 중심 → 산단 주변 지역 공급 없음
- 만보기 앱: 걸음만 → 사회적 연결 부재
- **산단파크 = 산단 특화 + 사회적 연결 + 게이미피케이션 + 지역경제**

## 3. What — 핵심 기능 (심사기준: 창의성)

### 3-1. 산단 RPG 맵
- 실제 산업단지를 게임 맵 스타일로 재구성
- 어트랙션 핀: 소셜다이닝 / 러닝 / 독서 / 클라이밍 / 플로깅 / 제휴카페 / 트라이브 아지트
- 핀 탭 → 바텀시트로 퀘스트 상세, 보상, 참여자 수, 거리 표시
- "내 위치" 펄스 애니메이션 (포켓몬GO 감성)

### 3-2. 퀘스트 시스템
**일일 퀘스트 (건강한 습관):**
- 🌅 7시 기상 인증 → +10 SDC
- 🥗 건강식 인증 → +15 SDC
- 🚶 걸어서 출근 (스텝당) → +1 SDC
- 🌿 플로깅 → +30 SDC

**소셜 퀘스트 (사람과의 연결):**
- 📚 독서토론 → +50 SDC
- 🍳 소셜다이닝 → +60 SDC
- 🧗 클라이밍 체험 → +40 SDC

**보너스:** 주간 미션 올클리어 → x10, 길드 퀘스트 → x2

### 3-3. 3단계 레벨 시스템
| 레벨 | 이름 | 잠금 해제 |
|---|---|---|
| Lv.1 | 견습생 | 가입 기본, 솔로 퀘스트 구경 |
| Lv.2 | 탐험가 | 소셜 퀘스트 참여, 부족 가입 |
| Lv.3 | 길드마스터 | 부족 개설, 이벤트 호스팅, 챌린지 주최 |

### 3-4. 트라이브 (온라인 빌리지)
- **에스토니아 시민권** 컨셉 — 온라인 빌리지 시티즌십
- 취향 기반 부족 매칭 (러닝/독서/쿠킹/AI 등)
- 산단별 부족 → 전국 **명량운동회** (올림픽처럼 랭킹 경쟁)
- 온라인 채팅 → 안전한 오프라인 아지트로 전환

### 3-5. 하이브리드 보상 체계
**획득 라인 (게이미피케이션):**
- SDC 코인 적립, XP 축적, 뱃지 해금, 부족 랭킹 상승

**지출 라인 (지역 상권 연계):**
- 제휴 카페 20% 할인, 소셜다이닝 참가권, 미용실 쿠폰
- 보조금을 "재미"로 포장한 지역경제 순환

### 3-6. Before → After 성장 기록
- 12주 전 vs 오늘 시각화 (친구 0명 → 24명, 이직 의향 "높음" → "낮음")
- 간증 스토리 아카이브 → 다음 신규 유저의 학습 자료

## 4. How — 구조 & 플로우 (심사기준: 효과성)

### 4-1. 사용자 저니 (12주)

**Week 1-2 (Lv.1 견습생):**
- 앱 설치 → 오늘의 퀘스트 눈으로 구경
- 솔로 퀘스트 한두 개 시도 (기상 인증 등)
- 눈팅만으로 "아, 여기 애들이 이렇게 노는구나"

**Week 3-6 (Lv.2 탐험가):**
- 첫 소셜 퀘스트 참여 (독서토론, 플로깅)
- 취향 매칭으로 첫 부족 가입
- 오프라인 아지트 첫 방문

**Week 7-12 (Lv.3 길드마스터):**
- 직접 챌린지 주최 (예: "수요 러닝크루")
- 부족 리더십, 멘토링
- 간증: "살만해요"

### 4-2. 전국 확장 플래닝
- Pilot: 구미 → Phase 1: 창원·완주·반월 → Phase 2: 10개 문화친화 산단
- 2027년 전국 1,330개 산단 중 100개 커버 목표
- 산단별 로컬 특성 반영 (구미=제조, 창원=기계, 완주=지방도시)

### 4-3. 기술 스택 (프로토타입 기준)
- Frontend: Next.js 16 (App Router) + Tailwind CSS v4
- 모바일 반응형 웹앱 (PWA 전환 가능)
- 향후: 만보기 연동(헬스킷/구글핏), QR 인증, 오픈 채팅

### 4-4. 지속가능성
- **아카이브형** — 일회성 이벤트가 아니라 계속 쌓이는 플랫폼
- **중소기업 포용** — 블라인드의 사각지대 공략
- **지자체-기업-유저 3자 공생** — 산단공단 기금 + 제휴처 할인 + 유저 참여

## 7. 성공 지표 (KPI)

**단기 (6개월):**
- DAU 500명, WAU 2,000명 (구미 파일럿)
- 주간 퀘스트 완료율 60%
- 부족 가입률 40%

**중기 (1년):**
- 제휴처 100곳, SDC 유통량 월 500만 포인트
- 외로움 지수 감소 (설문 기반)
- 이직률 비교 대조군 대비 감소

**장기 (3년):**
- 10개 문화친화 산단 표준 플랫폼
- 지자체 기금 자체 순환 구조 확립
